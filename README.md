## 📖 About Bharat Pulse AI

Bharat Pulse AI addresses the challenge of fragmented, delayed, and conflicting public service information.

The platform:

- Verifies information from multiple sources
- Detects contradictions in public service updates
- Generates confidence-based reliability scores
- Enables citizen reporting and validation
- Predicts potential service disruptions
- Provides area-level public service health insights
- Supports offline-first accessibility

## 🛠️ Run Locally (Optional)

### Prerequisites

- Node.js
- Gemini API Key

### Installation

1. Clone the repository

```bash
git clone <repository-url>
cd bharat-pulse-ai
```

2. Install dependencies

```bash
npm install
```

3. Configure environment variables

Create a `.env.local` file and add:

```env
GEMINI_API_KEY=your_gemini_api_key
```

4. Start development server

```bash
npm run dev
```

5. Open:

```text
http://localhost:3000
```
